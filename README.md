# Setup

1. Change $basedir in algm
1. Install node.js  
    Also `npm install zombie` in home dir.
2. For kattis submissions.  
    * Fetch a .kattisrc
3. For uva submissions.  
    * Update username in uvaSubmit
    * Change password file in uvaSubmit (.uvapass) or something
